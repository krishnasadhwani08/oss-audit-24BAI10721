Project Summary
  This project is an audit of VLC Media Player as an open-source software. It focuses on understanding open-source philosophy, licensing (GPL), and how software behaves inside a Linux environment. The project also includes practical shell scripting tasks to demonstrate Linux command-line skills and automation. 
Chosen Software
  VLC Media Player VLC is a free and open-source multimedia player that supports multiple file formats without requiring additional codecs. It is widely used due to its simplicity, flexibility, and independence from proprietary restrictions. 
Shell Scripts 
  Script 1 — System Identity Report Displays system details such as kernel version, user, home directory, uptime, and date/time. 
  Script 2 — FOSS Package Inspector Checks whether VLC is installed and displays version, license, and description. 
  Script 3 — Disk and Permission Auditor Analyzes important system directories and shows their permissions and disk usage. 
  Script 4 — Log File Analyzer Reads a log file, counts occurrences of a keyword, and displays matching lines. 
  Script 5 — Open Source Manifesto Generator Takes user input and generates a personalized open-source philosophy statement saved to a file.
